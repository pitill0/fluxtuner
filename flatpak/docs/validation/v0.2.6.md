## Environment notes

### CRUX Linux / Flatpak

On CRUX, local Flatpak builds should use:

```bash
flatpak-builder \
  --force-clean \
  --disable-rofiles-fuse \
  build-dir \
  io.github.pitill0.Fluxtuner.yml
```

Optional user install:

```bash
flatpak-builder \
  --force-clean \
  --disable-rofiles-fuse \
  --user \
  --install \
  build-dir \
  io.github.pitill0.Fluxtuner.yml
```

Reason:

- `--disable-rofiles-fuse` is required/recommended on the tested CRUX environment due to FUSE permission issues.

Clean local Flatpak build state:

```bash
flatpak list | grep -i fluxtuner
flatpak uninstall --user io.github.pitill0.Fluxtuner
flatpak uninstall --unused
rm -rf build-dir .flatpak-builder
```

If installed system-wide:

```bash
sudo flatpak uninstall --system io.github.pitill0.Fluxtuner
```

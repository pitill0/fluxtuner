# Changelog

## 0.1.0 - Initial public release candidate

### Fixed
- Theme selector previews are now temporary; moving through themes no longer changes the active theme until `Enter` is pressed.
- Leaving the theme selector restores the last applied theme if a preview was not confirmed.


- Textual TUI for internet radio playback.
- Table-based station browser with named columns for search, favorites, history and playlist stations.
- Radio Browser search with live debounce and local cache.
- Playback through `mpv` with JSON IPC controls.
- Favorites with custom names and user tags.
- Dynamic tag playlists and persistent manual playlists.
- Recently played history.
- Theme system with bundled themes and in-app preview/save flow.
- Visual Now Playing panel with active station marker and volume bar.
- CLI helpers for import/export, theme management and cache cleanup.

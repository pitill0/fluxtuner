## Summary

FluxTuner `v0.2.9` is a small patch release focused on stabilizing the Textual TUI station details side panel.

This release fixes station details rendering and side-panel layout after line breaks, preventing visual glitches in the TUI details panel.

The release also aligns package version metadata, changelog entries, AppStream release metadata and the Flathub manifest source pin.

## Changes validated

### TUI station details side panel

Validated change:

- Fixed station details rendering after line breaks.
- Fixed side-panel layout stability when station metadata wraps across multiple lines.
- Confirmed the details panel remains readable after selecting stations with longer names, tags or metadata.
- Confirmed the station table and side-panel layout remain coherent after moving between stations.
- Confirmed playback markers remain unaffected by the side-panel rendering fix.

## Summary

FluxTuner `v0.2.8` is a small GTK GUI polish release.

It improves the side-panel organization and makes active view status messages clearer when switching between Search results, Favorites, History and tag playlists.

---

## Changes validated

### GTK GUI Library navigation

Validated change:

- Added a dedicated **Library** section to the GTK GUI side panel.
- Moved navigation actions into **Library**:
  - `Show favorites`
  - `History`
- Kept **Favorites** focused on favorite editing actions:
  - Add favorite
  - Remove favorite
  - Edit favorite name
  - Edit favorite tags
  - Save edited favorite
- Confirmed **Playlists** remains separate.

Expected layout:

```text
Library
- Show favorites
- History

Favorites
- Add favorite
- Remove favorite
- Edit favorite name
- Edit favorite tags
- Save edited favorite

Playlists
- Favorite tag
- Show
- Random
- Tags
- Clear
```

### GTK GUI active view status

Validated change:

- Added consistent status messages for list-like views.
- Search now reports a clear Search results status.
- Favorites now reports a clear Favorites status.
- History now reports a clear History status.
- Tag playlist now reports a clear Tag playlist status.

Expected examples:

```text
Search results · N stations
Favorites · N stations
History · N stations
Tag playlist · <tag> · N stations
```
